
NAME
    New-UDDataGrid
    
SYNOPSIS
    Displays data in a table-style grid.
    
    
SYNTAX
    New-UDDataGrid [[-Id] <String>] [[-Columns] <Hashtable[]>] [[-LoadRows] <Endpoint>] [[-Height] <String>] [-AutoHeight] [-AutoPageSize] [-CheckboxSelection] [-CheckboxSelectionVisibleOnly] [[-ColumnBuffer] <Int32>] [[-ColumnThreshold] <Int32>] [[-Density] <String>] [[-PageSize] <Int32>] [[-RowsPerPageOptions] <Int32[]>] [-Pagination] [[-Language] <String>] [[-LoadDetailContent] <Endpoint>] [[-DetailHeight] <String>] [[-OnEdit] <Endpoint>] [[-OnExport] <Endpoint>] [-ShowQuickFilter] [<CommonParameters>]
    
    
DESCRIPTION
    Displays data in a table-style grid. Provides support for sorting, paging, and filtering of large data sets.
