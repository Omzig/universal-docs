# Import-PSUSecret

## SYNOPSIS
Imports secret from configured secret vaults. 

## SYNTAX

```
Import-PSUSecret [-ComputerName <String>] [-AppToken <String>] [-UseDefaultCredentials] [-Integrated]
 [<CommonParameters>]
```

## DESCRIPTION
Imports secret from configured secret vaults. 

## EXAMPLES

### Example 1
```powershell
PS C:\> Import-PSUSecret
```
