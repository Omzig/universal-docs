
NAME
    New-UDTextbox
    
SYNOPSIS
    Creates a textbox.
    
    
SYNTAX
    New-UDTextbox [[-Id] <String>] [[-Label] <String>] [[-Placeholder] <String>] [[-Value] <String>] [[-Type] <String>] [-Disabled] [[-Icon] <Object>] [-Autofocus] [-Multiline] [[-Rows] <Int32>] [[-RowsMax] <Int32>] [-FullWidth] [[-Mask] <String>] [-Unmask] [[-Variant] <String>] [[-ClassName] <String>] [[-OnEnter] <Endpoint>] [[-OnBlur] <Endpoint>] [-Shrink] [<CommonParameters>]
    
    
DESCRIPTION
    Creates a textbox. Textboxes can be used by themselves or within a New-UDForm.
