# Get-PSUEndpointDocumentation

## SYNOPSIS
Returns endpoint help documentation definitions.

## SYNTAX

```
Get-PSUEndpointDocumentation [-ComputerName <String>] [-AppToken <String>] [-UseDefaultCredentials]
 [-Integrated] [<CommonParameters>]
```

## DESCRIPTION
Returns endpoint help documentation definitions.

## EXAMPLES

### Example 1
```powershell
PS C:\> Get-PSUEndpointDocumentation
```

Returns all help documentation definitions.
