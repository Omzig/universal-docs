# Get-PSUComputer

## SYNOPSIS
Returns the agent computers registered with PowerShell Universal.

## SYNTAX

### All (Default)
```
Get-PSUComputer [-ComputerName <String>] [-AppToken <String>] [-UseDefaultCredentials] [-Integrated]
 [<CommonParameters>]
```

### Id
```
Get-PSUComputer [-Id] <Int64> [-ComputerName <String>] [-AppToken <String>] [-UseDefaultCredentials]
 [-Integrated] [<CommonParameters>]
```

## DESCRIPTION
Returns the agent computers registered with UA.

## EXAMPLES

### Example 1
```
PS C:\> Get-UAComputer
```

Returns all the job agent computers connected to UA.
