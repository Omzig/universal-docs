# Markdown

`New-UDMarkdown` accepts a markdown string and renders it as HTML elements within a dashboard.&#x20;

```powershell
New-UDMarkdown -Markdown "
   # Header
   - List Item 1
   - List Item 2
   
   ## Sub Header
"
```

## API

* New-UDMarkdown
